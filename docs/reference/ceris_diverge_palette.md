# CERIS diverging color palette for correlation heatmaps

CERIS diverging color palette for correlation heatmaps

## Usage

``` r
ceris_diverge_palette(n = 26L)
```

## Arguments

- n:

  Number of colors (default 26)

## Value

Character vector of hex colors

## Examples

``` r
pal <- ceris_diverge_palette(11)
length(pal)
#> [1] 11
```
